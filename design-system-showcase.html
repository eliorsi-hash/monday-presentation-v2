Design Rules

CSS lockdown blocks in design-system.css enforce these at runtime
Two Font Weights Only
Import only wght@400;600. The system uses Regular (400) for body and most headings, SemiBold (600) for h1 and emphasis.

Never use 300, 500, or 700+. Those tokens are aliased to 400/600 in design-system.css.
Zero Letter Spacing
All --tracking-* tokens resolve to 0. Poppins already ships at the right optical tracking.

Never set a custom letter-spacing on any slide element. The lockdown block enforces letter-spacing: normal !important on all .slide-container *.
No Italics
Italic is not part of the system. em, i, .italic, and any element with an inline font-style:italic are all forced upright via font-style: normal !important.

Use .highlight or .highlight--bold for emphasis instead.
One Highlight Per Headline
.highlight — color only (yellow / purple). Weight stays as-is.
.highlight--bold — weight only (600). Color stays as-is.

Never combine both on the same word. Choose one axis — color or weight, never both.
No Dividers, No Uppercase
Never add horizontal rules or decorative separator lines between slide sections — whitespace and typographic hierarchy carry the structure.

Never use text-transform: uppercase on headings. Poppins is designed for sentence or title case only.
Poppins Everywhere
The type lockdown block forces font-family: 'Poppins', sans-serif !important on every element. No serif fallbacks, no system fonts.

Only code, pre, and .mono elements use SF Mono / Fira Code.
