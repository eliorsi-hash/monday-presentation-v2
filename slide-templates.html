Slide Templates

All reusable CSS template classes + component patterns. Each class defines the layout for one .slide-container. Components (stat blocks, step cards, charts, tables, timelines) drop in as children. This is the single source of truth for all slide layouts.

Usage

Each slide uses exactly one template class. All classes are defined in design-system.css — just inline the CSS file and apply the class.

/* Inline design-system.css, then: */
<div class="slide-container tmpl-cover slide-1" data-slide-index="0">
  <!-- centered content -->
</div>

<div class="slide-container tmpl-twocol slide-2" data-slide-index="1">
  <div class="left-col">...</div>
  <div class="right-col">...</div>
</div>

<div class="slide-container tmpl-stats slide-3" data-slide-index="2">
  <div class="slide-header">...</div>
  <div class="stats-row"><!-- stat-block x 4 --></div>
</div>
