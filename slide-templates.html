Usage

Each slide uses exactly one template class. All 8 classes are defined in design-system.css — just inline the CSS file and apply the class.

/* Inline design-system.css, then: */
<div class="slide-container tmpl-cover slide-1" data-slide-index="0">
  <!-- centered content -->
</div>

<div class="slide-container tmpl-twocol slide-2" data-slide-index="1">
  <div class="left-col">...</div>
  <div class="right-col">...</div>
</div>

<div class="slide-container tmpl-stats slide-3" data-slide-index="2">
  <div class="slide-header">...</div>
  <div class="stats-row"><!-- stat-block × 4 --></div>
</div>

/* Slide hide/show (already in design-system.css): */
.slide-container:not(.slide-active) { display: none !important; }
